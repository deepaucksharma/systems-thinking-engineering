# Incentive Death Spiral

## Definition

A V2 cross-layer causal loop ($A \to P \to E$) where broken organizational incentives degrade people's health, which drives down execution quality, spawning more fires that reinforce the broken incentives.

## Mechanism

```
Heroics Rewarded → Prevention Invisible → People Optimize for Heroics
→ Problems Not Fixed → More Heroics Required → Burnout
→ Sustainability (Su) ↓ → Quality (Q) ↓ → More Fires → More Heroics
```

## Intervention

Breaking this loop requires changing what gets visibly rewarded. It **cannot** be fixed at the team level (Execution block). The lever is purely within [Org Alignment (Block A)](../legacy/org-alignment.md) via Incentive Alignment ($IA$).
