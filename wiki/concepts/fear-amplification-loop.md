# Fear Amplification Loop

## Definition

A V2 cross-layer causal loop ($P \to L \to V$) where low psychological safety destroys the ability to learn, passing false signal upward, which pollutes strategic decisions.

## Mechanism

```
Fear of Bad News → Problems Hidden → Feedback Distorted
→ Strategy Based on False Signal → Wrong Bets Made
→ Outcomes Fail → Blame Intensifies → More Fear
```

## Intervention

Breaking this requires a blameless culture and leader modeling of truth-telling. It **cannot** be fixed by implementing faster feedback. In this state, an increase in [Feedback Speed](feedback-speed.md) without fixing Psychological Safety ($PS$) only accelerates fear-driven distortion.
