# Sustainability Drain Loop

## Definition

A V2 cross-layer causal loop ($P \to E \to A$) where continuous overload draws down future capacity to pay for today's output, eventually collapsing the system.

## Mechanism

```
Overload → Burnout Accumulates → Quality ↓ + Learning Stops
→ Debt Accumulates → More Firefighting Required → More Overload
```

## Intervention

Breaking requires genuine load reduction (cutting scope) plus structural slack protection. It **cannot** be fixed by efficiency improvements alone.
