# Source: Canonical V2.1 Architecture

**Raw File:** `raw/6.md`

## Summary
The definitive, canonical restructuring of the control-theory model. This source explicitly formalizes the 5-Block framework ($V, P, E, A, L$) and introduces hard mathematical and conceptual bounds derived from queuing physics (Batch Size, Tail Risk), cognitive load decomposition (Intrinsic, Extraneous, Germane), and Cynefin domain awareness. It establishes the final, 6-part editorial taxonomy for the wiki (`THEORY`, `BLOCKS`, `DIAGNOSIS`, `PLAYBOOK`, `MEASUREMENT`, `REFERENCE`).

## Key Advancements over V1/V2
1. **Queuing Physics in Execution (E):** Formalizes that Focus/WIP is insufficient without controlling Batch Size and measuring Tail-Risk (Obesity Index) rather than averages.
2. **Measurement Reliability Theorem:** Formal assertion that all metrics are effectively gated by Psychological Safety ($PS$). Without Safety, quantitative signals turn into political noise.
3. **P-Block Decomposition:** The SPACE framework is integrated as the primary measurement for the People System. Cognitive capacity is decomposed into Extraneous Load ($EL$) vs. Germane Capacity ($GC$).
4. **Time Constants (Nyquist):** Strict, layer-specific time constants define how long an intervention must run before extracting signal (stopping Manager Oscillation).
